Tree Like Data Structure Visualiser

Binary Search Tree

A hierarchal data structure used for efficient searching, insertion and deletion.
Each node has a key, a left child and a right child.
Where each left child must be smaller than the node, and each right child must be greater than the node.

AVL Tree

A self-balancing Binary Search Trees.
AVL trees have the same properties as a BST with the addition of a balance factor to ensure operations are efficient.

Binary Heap

A hierarchal structure where the most important node is always stored at the root of the heap.
Each node must follow the heap priority which is a set of rules defined by the type of heap.
This visualiser includes a Min Heap where the smallest node is kept at the root, and a Max Heap, where the largest node is kept at the root.

Red Black Tree

Red Black trees have the same properties as a BST with the additon of node colouring.
Nodes can be coloured Red or Black, and this is used along with rotations to sort the tree into a more efficient form.

Fibonacci Heap

A collection of connected Min Heaps where the heap is sorted only after a node is removed.
