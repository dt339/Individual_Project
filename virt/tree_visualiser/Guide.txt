---To Run the Application---

Open terminal
Navigate to Folder Individual_Project
Run 'virt\Scripts\activate.bat' to start virtual environment
---If this fails, Django is probably not installed. 'py -m pip install Django' for windows --
Navigate to the outer Tree_Visualiser folder (inside of virt)
Run 'py manage.py runserver' to start the server
http://127.0.0.1:8000/home/
